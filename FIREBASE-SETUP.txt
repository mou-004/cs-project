FAST FIREBASE EMAIL VERIFICATION SETUP

Project: csproject-24094

1. Firebase Console > Authentication > Sign-in method.
2. Enable Email/Password.
3. Authentication > Templates > Email address verification.
4. Configure the verification email template.
5. Authentication > Settings > Authorized domains.
6. Add localhost and your deployed domain.
7. Run using VS Code Live Server or python -m http.server 5500.

This build uses Firebase Identity Toolkit REST directly.
It does not download firebase-app, firebase-auth, or firebase-firestore SDK files.

Customer login requires emailVerified=true.
No local Customer login fallback is available.
